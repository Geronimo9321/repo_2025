Proyectos

En esta sección podrán ver algunos de mis proyectos desarrollados, de manera individual y junto a compañeros de distintos cursos que fui realizando durante este tiempo.

Proyecto 1: tienda de Cheesecake

Esta es una simulación de una tienda online para una pastelería especializada en la elaboración de productos tipo cheesecake. La página fue desarrollada con HTML, CSS y JavaScript, y cuenta con sus respectivas secciones, un carrusel y todas las funcionalidades básicas de una tienda online. En la sección de contacto se detallan los medios de comunicación, así como un formulario para reclamos o sugerencias. Además, desde la misma página se puede simular una compra. Para ver más, ingresá en Ver Proyecto.

Proyecto 2: Registro de Personas

Esta es una simulación de una página web diseñada para registrar personas, donde además se pueden visualizar perfiles destacados y figuras relevantes. La página fue desarrollada con HTML, CSS y JavaScript, y cuenta con un diseño claro y funcional para la gestión de registros. Este proyecto se encuentra disponible como repositorio en GitHub, y está pensado como ejemplo práctico de manipulación de datos desde el front-end. Para ver más, ingresá en Ver Proyecto.

Proyecto 3: Aplicacion de Registro hecho con Tkinter

Esta es una aplicación de escritorio desarrollada con Python utilizando la librería gráfica Tkinter, una herramienta integrada que permite crear interfaces gráficas de usuario (GUI) de forma sencilla y eficiente. La aplicación está diseñada para el registro de personas, con la finalidad de controlar los datos de una persona, donde se puede agregar, modificar, visualizar o eliminar información relevante a través de una interfaz intuitiva. Este proyecto se encuentra disponible como repositorio en GitHub, y representa un ejemplo práctico de desarrollo de aplicaciones de escritorio con Python. Para ver más, ingresá en Ver Proyecto.
